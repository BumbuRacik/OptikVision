# OptikVision

Aplikasi Transaksi Penjualan Optik berbasis Java Swing dan MySQL.

## Fitur

- Data Pasien
- Resep Mata
- Pilih Frame
- Pilih Lensa
- Hitung Total Otomatis
- Simpan ke Database MySQL
- Cetak Nota
- CRUD Transaksi

## Teknologi

- Java Swing
- JDBC
- MySQL
- NetBeans IDE
- FlatLaf

## Database

optik_db

Tabel:

- transaksi
- frame
- lensa
